## 4. Промокоды и скидки

| №   | Проверка | Статус | Фактический результат |
|---|---|---|---|
| 4.1 | Ввод валидного промокода | PASS | Скидка применилась, сумма уменьшилась |
| 4.2 | Ввод невалидного промокода | FAIL | Страница обновилась, сообщения об ошибке нет. Промокод не применился, но пользователь не уведомлён |
| 4.3 | Ввод истёкшего промокода | PASS | Промокод не принят, сообщение об ошибке есть |
| 4.4 | Отмена промокода | PASS | Скидка убрана, сумма вернулась к исходной |
| 4.5 | Перерасчёт суммы при изменении корзины с активным промокодом | PASS | Сумма пересчиталась с учётом скидки |
